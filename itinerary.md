## 📅 五日行程詳述

### Day 1: 高雄 / 福岡
*   **摘要**：抵達九州第一大城 - 福岡，開啟精彩旅程。
*   **航班**：長榮航空 BR120 (15:30 高雄 -> 19:20 福岡)
*   **主要景點**：
    *   **福岡機場**：距離市區極近，交通便利。
*   **在地推薦**：
    *   [美食] 博多拉麵 一蘭 (機場店) - 最後一碗道地拉麵的機會。
    *   [伴手禮] 博多通りもん - 福岡最受歡迎的伴手禮。
    *   [伴手禮] 福砂屋 長崎蛋糕 - 經典底部砂糖顆粒蛋糕。
*   **用餐**：晚 - 機上簡餐
*   **住宿**：QUINTESSA福岡天神南 (QUINTESSA HOTEL FUKUOKA TENJIN MINAMI)

### Day 2: 水鄉柳川與熊本名城
*   **摘要**：體驗柳川扁舟搖船，參訪日本三大名城之一的熊本城。
*   **主要景點**：
    *   **柳川扁舟搖船**：日本威尼斯，聆聽船夫吟唱。
    *   **熊本城**：日本三大名城，壯觀的「武者返」石牆。
    *   **櫻之馬場城彩苑**：重現江戶時代城下町風情。
    *   **萌熊電鐵**：可愛的萌熊彩繪電車。
*   **在地推薦**：
    *   [美食] 元祖本吉屋 - 柳川最古老的蒸鰻魚飯老店。
    *   [美食] 菅乃屋 馬肉可樂餅 - 熊本特色馬肉料理。
    *   [伴手禮] 譽之陣太鼓 - 熊本代表性紅豆甜點。
*   **用餐**：早 - 飯店 / 午 - 日式風味御膳 / 晚 - 飯店百匯 (和牛無限享用)
*   **住宿**：南關溫泉 SEKIA (SEKIA)

### Day 3: 阿蘇大觀峰與由布院散策
*   **摘要**：探訪白川水源，漫步絕美金鱗湖與歐洲童話村。
*   **主要景點**：
    *   **白川水源**：名水百選，水質清澈甘甜。
    *   **阿蘇大觀峰**：360度俯瞰阿蘇五岳「涅槃像」。
    *   **湯布院/湯之坪街道**：女性票選最愛度假勝地。
    *   **湯之坪歐洲童話村**：迷你主題樂園，置身童話世界。
    *   **金鱗湖**：溫泉與清水湧出，晨霧如夢似幻。
*   **在地推薦**：
    *   [美食] B-speak 奶油捲 - 湯布院超人氣綿密甜點。
    *   [美食] 金賞可樂餅 - 外皮酥脆內餡濃郁。
    *   [伴手禮] Milch 半熟起司蛋糕 - 濃郁起司香氣。
*   **用餐**：早 - 飯店 / 午 - 湯布院蒸籠料理 / 晚 - 飯店自助餐或會席料理
*   **住宿**：GRANDVRIO HOTEL BEPPUWAN WAKURA (グランヴィリオホテル別府灣和蔵)

### Day 4: 宇佐神宮與門司港夜景
*   **摘要**：參拜國寶宇佐神宮，欣賞皿倉山新日本三大夜景。
*   **主要景點**：
    *   **宇佐神宮**：全日本八幡宮總本宮，國寶級建築。
    *   **門司港懷舊散策**：明治大正時期西式建築，異國情調。
    *   **皿倉山纜車**：新日本三大夜景，價值百億美元。
*   **在地推薦**：
    *   [美食] 伽哩本舖 - 門司港名物燒咖哩。
    *   [伴手禮] 門司港香蕉巧克力餅乾 - 香蕉拍賣發源地特色商品。
*   **用餐**：早 - 飯店 / 午 - 北九州燒咖哩 / 晚 - 自理 (發 ¥1000)
*   **住宿**：Premier 門司港飯店 (PREMIER HOTEL MOJIKO)

### Day 5: 太宰府天滿宮與賦歸
*   **摘要**：參拜學問之神，搭乘旅人觀光列車，帶著回憶返台。
*   **主要景點**：
    *   **太宰府天滿宮**：祭祀學問之神，祈求金榜題名。
    *   **太宰府星巴克**：建築大師隈研吾設計，木造幾何美學。
    *   **太宰府旅人觀光列車**：和風美感開運列車。
*   **在地推薦**：
    *   [美食] かさの家 梅枝餅 - 參道名店，現烤美味。
    *   [店家] 太宰府星巴克 - 全球最美星巴克之一。
    *   [伴手禮] 明太子 - 福岡代表特產，推薦老店「ふくや」。
*   **用餐**：早 - 飯店 / 午 - 螃蟹會席料理 / 晚 - 機上簡餐
*   **航班**：長榮航空 BR119 (20:20 福岡 -> 22:10 高雄)

## 📝 實用旅遊筆記

### 退稅須知
*   **消耗品 (食品/藥妝)**：5,000 ~ 500,000 日圓，須封裝且不可在日本境內拆封。
*   **一般品 (衣物/電器)**：5,000 日圓以上，可拆封使用。
*   **退稅門檻**：同日、同一商店累計消費滿 5,000 日圓 (未稅)。

### 行前叮嚀
*   **電壓**：100V，插頭與台灣相同。
*   **小費**：導遊領隊小費每日 NTD 300。
*   **行李**：托運 23KG，手提 7KG。
*   **SIM 卡**：每日 2GB 高速，需開啟數據漫遊。
